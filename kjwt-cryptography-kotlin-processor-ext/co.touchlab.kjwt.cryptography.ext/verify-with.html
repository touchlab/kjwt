suspend fun JwtParserBuilder.verifyWith(algorithm: SigningAlgorithm.EdDSABased, jwk: Jwk.Okp, keyId: String? = jwk.kid, cryptoProvider: CryptographyProvider = CryptographyProvider.Default): JwtParserBuilder(source)

Configures the parser to verify JWS signatures using an EdDSA public key derived from the given Jwk.Okp JWK.

Return

this builder for chaining

Parameters

algorithm

the EdDSA-based signing algorithm (Ed25519 or Ed448)

jwk

the OKP JWK containing the public key parameter x and curve crv

keyId

optional key ID override; when set, the parser will only use this key if the token's kid header matches. Defaults to the JWK's own kid field.

cryptoProvider

the CryptographyProvider used to decode the key; defaults to CryptographyProvider.Default
