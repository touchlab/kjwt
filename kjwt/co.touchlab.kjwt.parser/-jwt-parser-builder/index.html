JwtParserBuilder

class JwtParserBuilder(jsonInstance: Json)(source)

Configures and builds a JwtParser.

Example:

val signingKey = SigningAlgorithm.HS256.newKey()
val parser = Jwt.parser()
    .verifyWith(signingKey)
    .requireIssuer("myapp")
    .clockSkew(30L)
    .build()
val jws = parser.parse(token)
