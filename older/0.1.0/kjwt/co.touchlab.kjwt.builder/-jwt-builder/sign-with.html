suspend fun <PublicKey : Key, PrivateKey : Key> signWith(algorithm: SigningAlgorithm<PublicKey, PrivateKey>, registry: JwtKeyRegistry, keyId: String? = null): JwtInstance.Jws(source)

Looks up the private key from registry and builds a JWS compact serialization.

The registry is searched using algorithm and keyId as the look-up criteria (see JwtKeyRegistry for the full look-up order). If no matching key is found an IllegalStateException is thrown.

Passing co.touchlab.kjwt.model.algorithm.SigningAlgorithm.None delegates directly to build (unsecured token) without consulting the registry.

Return

the resulting JwtInstance.Jws compact serialization

Parameters

algorithm

the signing algorithm to use

registry

the key registry to look up the private key from

keyId

optional key ID used for registry look-up and embedded in the JWT header's kid field. Defaults to null.

See also

JwtKeyRegistry

Throws

IllegalStateException

if no signing key for algorithm (and keyId) is found in registry
